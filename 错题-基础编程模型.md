## 将一个正整数用二进制表示并转换为一个String

- **除2取余循环商，逆序排列法**:用2整除十进制整数，可以得到一个商和余数；再用2去除商，又会得到一个商和余数，如此进行，直到商为0时为止，然后把先得到的余数作为二进制数的低位有效位，后得到的余数作为二进制数的高位有效位，依次排列起来。
- 算法分析

  - 字符串相加时可以通过加号顺序实现逆序排列
  - n = n/2实现取商
  - n % 2 实现取余

```java
        Scanner in = new Scanner(System.in);
        int N = in.nextInt();
        String s = "";

        for (int n = N; n > 0; n /= 2) {
            s = (n % 2) + s;
        }
```
